# Amazing - 大模型管理平台

基于 Agent-Teams 协同开发范式的企业级大模型管理平台。

## 🚀 快速开始

```bash
# 初始化项目
amazing init

# 选择角色
amazing role select

# 启动开发
make dev
```

## 📁 项目结构

```
amazing/
├── .agents/              # Agent 定义
├── .claude/             # Claude Code 配置
├── backend/             # 后端服务
├── frontend/            # 前端应用
├── infra/               # 基础设施
└── docs/                # 文档
```
