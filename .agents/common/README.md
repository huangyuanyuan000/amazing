# Common Agent - 通用模块

负责用户管理、权限控制、日志审计等通用功能。

## Sub-Agents

- **PM**: 需求分析、PRD 编写
- **Frontend**: UI/UX 实现
- **Backend**: API 开发
- **QA**: 测试用例、自动化测试
- **Ops**: 部署、监控
- **Operation**: 数据分析、配置管理

## Skills

- `user-management`: 用户 CRUD、角色管理
- `auth`: 认证授权 (JWT/OAuth2)
- `logging`: 日志收集、审计
- `api-design`: RESTful API 设计

## 进化指标

- 代码质量分数
- 测试覆盖率
- API 响应时间
