# Config Management Skill - 运营配置管理

## 功能描述
提供运营配置管理规范、功能开关实现和配置变更流程。

## 触发方式
- 运营配置变更
- 功能开关管理
- A/B 测试配置

## 核心内容

### 1. 配置分类
| 类型 | 示例 | 变更频率 |
|------|------|----------|
| 系统配置 | 数据库连接、缓存 | 低 |
| 业务配置 | 费率、限额 | 中 |
| 功能开关 | 新功能灰度 | 高 |
| 运营配置 | 活动、推荐 | 高 |

### 2. 功能开关
```python
feature_flags = {
    "new_checkout": {
        "enabled": True,
        "rollout_percentage": 10,
        "whitelist": ["user_001"],
    }
}
```

### 3. 配置变更流程
1. 提交变更申请 → 2. 审批 → 3. 测试环境验证 → 4. 生产发布 → 5. 效果监控

### 4. 配置安全
- 敏感配置加密存储
- 配置变更审计日志
- 配置版本管理（支持回滚）
- 配置生效确认机制

## 示例
```json
{
  "change_id": "CFG-001",
  "config_key": "payment.max_amount",
  "old_value": 10000,
  "new_value": 50000,
  "reason": "提升单笔支付限额",
  "approver": "admin"
}
```

## 进化能力
- 配置模板持续扩充
- 变更影响自动评估
- 配置冲突自动检测
